## Const for Hello World of ES2015 in Node Core

```
const foo = "foo";
foo = "bar"; // should not assign
console.log(foo);
```

```
$ node const.js
foo
```

### Example 3

```
"use strict";
const foo = "foo";
let scope1 = (variable) => {
    console.log(variable);
}
let scope2 = () => {
    const foo = "bar";
    console.log(foo)
}
console.log(foo);
scope1(foo);
scope2();
```

```
$ node const.js
foo
foo
bar
```
